# Changelog

## 1.0.0-beta.1 - 2026-04-05
- Modernized the export pipeline around a shared feature model.
- Added direct `CSV` and `GeoJSON` export from the mod.
- Added bundled `GeoSkylines.ExportCli` for `SHP` and `GeoParquet`.
- Added batch export via `Right Ctrl + E`.
- Added macOS development and build support.
- Added release packaging scripts, example config, and sample export files.
- Kept legacy `import_export.txt` support while moving the canonical config name to `import_export.conf`.
